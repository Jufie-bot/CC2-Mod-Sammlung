# ??? Sandkasten

Hier findest du wiederverwendbare Bausteine, Skriptvorlagen und originale Referenzdateien für deine CC2 Mods.

## Struktur
- **/vanilla-referenz**: Enthält originale Lua und XML Dateien aus dem Game, entpackt als Referenz.
- **/ids**: Listen mit bekannten Fahrzeug-IDs, Waffen-IDs und Sound-IDs.
- **/Sound Collections**: Eigene Sound-Assets für Audio-Mods.
- **/snippets**: Kurze Code-Bausteine (Lua & XML) zum Herauskopieren.
- **/templates**: Gerüste für neue Mods (z.B. leere Mod-Strukturen).
