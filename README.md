## 🎯 Vision: Die Ultimative Mod-Entwicklung

> **"Ich analysiere nicht nur – ich erschaffe neu."**

Herzlich willkommen in meinem Forschungslabor. Dieses Repository ist weit mehr als nur eine Sammlung von existierenden Mods.

### 🧬 Mein Ziel
Ich arbeite an einer **eigenen, umfassenden Modifikation** für Carrier Command 2. Um das bestmögliche Ergebnis zu erzielen, gehe ich wissenschaftlich vor:
1.  **Sammeln**: Ich archiviere fast alle verfügbaren Mods.
2.  **Dekonstruieren**: Ich analysiere den Quellcode (Lua & XML), um zu verstehen, *wie* andere Entwickler Probleme gelöst haben.
3.  **Lernen & Dokumentieren**: Jede Erkenntnis landet in diesem Repository – als Wissen für mich und die gesamte Community.
4.  **Erschaffen**: Mit diesem Wissen baue ich meine eigene Vision.
