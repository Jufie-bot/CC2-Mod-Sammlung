# Security Policy

## Supported Versions

Wir unterstützen aktuell nur die neueste Version der Mod-Sammlung im `main` Branch.

## Reporting a Vulnerability

Solltest du eine Sicherheitslücke finden (z.B. in den Lua-Skripten), melde diese bitte nicht öffentlich über Issues.

Bitte kontaktiere uns stattdessen direkt über GitHub Discussions oder per E-Mail (falls hinterlegt). Wir werden uns so schnell wie möglich um eine Lösung kümmern.

Vielen Dank, dass du hilfst, die Community sicher zu halten!
