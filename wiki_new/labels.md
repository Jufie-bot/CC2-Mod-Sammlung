# GitHub Labels

Da Labels nicht automatisch per Datei synchronisiert werden, hier eine Liste der Labels, die wir verwenden wollen. Du kannst diese unter `Issues > Labels` anlegen.

- `bug` (Rot): Etwas funktioniert nicht.
- `enhancement` (Blau): Neues Feature oder Request.
- `documentation` (Grün): Änderungen an Wiki oder Readme.
- `help wanted` (Gelb): Unterstützung benötigt.
- `good first issue` (Lila): Gut für Einsteiger.
