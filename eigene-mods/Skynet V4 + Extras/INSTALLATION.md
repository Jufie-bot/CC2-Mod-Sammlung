# Skynet V4 - Installations-Anleitung

Um die **Skynet V4 + Extras** Mod zu installieren, folge diesen Schritten:

1.  Öffne deinen Windows-Explorer und navigiere zu folgendem Pfad:
    `%appdata%\Carrier Command 2\mods`
2.  Falls der Ordner `mods` noch nicht existiert, erstelle ihn.
3.  Kopiere den gesamten Ordner `Skynet V4 + Extras` direkt in das `mods`-Verzeichnis.
4.  Stelle sicher, dass die Struktur wie folgt aussieht:
    `%appdata%\Carrier Command 2\mods\Skynet V4 + Extras\mod.xml`
    `%appdata%\Carrier Command 2\mods\Skynet V4 + Extras\thumbnail.png`
    `%appdata%\Carrier Command 2\mods\Skynet V4 + Extras\content/...`

Die Mod wird beim Start des Spiels automatisch erkannt. Überprüfe das In-Game-Log oder die Konsole auf die Nachricht:
*"Skynet V4: Hook in holomap_screen erfolgreich."*, um sicherzustellen, dass alles funktioniert.
