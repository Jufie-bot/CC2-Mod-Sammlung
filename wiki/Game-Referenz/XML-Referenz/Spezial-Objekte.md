# Spezial-Objekte & Logistik

Hier werden Objekte gelistet, die spezielle Funktionen im Spiel übernehmen oder unendliche Vorräte besitzen.

## System-Einheiten & Logistik-Fähigkeiten

| Einheit | Versorgen | Lufttransport | Starten von... |
| :--- | :--- | :--- | :--- |
| **Carrier** | Schiffe (Treibstoff/Munition) | - | Luft-/Boden-Einheiten, Rettungsboot |
| **Mule** | Bodeneinheiten (Munition/Treibstoff) | - | - |
| **Petrel** | - | Bodeneinheiten | - |
| **Drydock** | - | - | Carrier |

## Gegenstände mit unendlichem Vorrat
Bestimmte Attachments werden im Spiel nicht als verbrauchbare Items im Lager geführt, sondern stehen (meist auf dem Carrier) unbegrenzt zur Verfügung.

| Item | Besonderheit | Benötigt Munition |
| :--- | :--- | :--- |
| **15mm Turret** | Unendlicher Vorrat | Nutzt 30mm Magazine |
| **160mm Gun** | Unendlicher Vorrat | - |
| **Naval Torpedo Launcher** | Unendlicher Vorrat | - |
| **Naval Camera** | Unendlicher Vorrat | - |
| **Naval Countermeasures** | Unendlicher Vorrat | - |
| **Cruise Missile Launcher** | Unendlicher Vorrat | - |

> [!TIP]
> Durch Anpassung der Lua-Skripte können Items im Loadout-Screen oft auch dann montiert werden, wenn der tatsächliche Bestand 0 ist (interessant für Modder).

## Rettungsboot (Lifeboat, Index 57)
Das kleine orangefarbene Rettungsboot auf der Backbordseite des Carriers. 
*   Es kann via Skript gestartet werden.
*   Es kann (derzeit) nicht manuell gesteuert werden.
